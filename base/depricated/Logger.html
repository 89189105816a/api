class Logger(builtins.object)
    Logger(heap: list, telegram_token: str, chat_id: str)
 
Класс логера - результаты успешной/не успешной работы записываем в базу, отправляем уведомление в telegram чат.
Чтобы не отправлять в базу данные после отработки каждого скрипта заливаем все целиком. Входные данные в логер
выглядят так [[результаты отработки скриптов по клиенту 1], [результаты отработки скриптов по клиенту 2], []...]
